# 🌊 Dream Journal

> *Dive into what you almost forgot.*

A beautifully designed dream tracking app built with React. Capture your dreams and nightmares the moment you wake up — before they drift away into the deep.

-----

## ✨ Features

- **Personalized welcome** — enter your name on first launch, greeted every time you return
- **Quick capture** — record dreams fast before they fade, with a title, full description, type, and tags
- **6 dream types** — Magical, Nightmare, Surreal, Lucid, Vague, Emotional — each with its own color
- **Filter & browse** — filter your dream history by type
- **Stats dashboard** — see your total dreams, nightmares, and lucid dreams at a glance
- **Persistent storage** — your dreams are saved across sessions, nothing is ever lost
- **Deep sea abyss aesthetic** — bioluminescent smoky glow, animated floating particles, and light rays filtering through dark water

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/dream-journal.git

# Navigate into the project
cd dream-journal

# Install dependencies
npm install

# Start the development server
npm run dev
```

### Usage

1. Open the app — you’ll be asked to enter your name on first launch
1. Tap **”+ Record a Dream”** right after waking up
1. Write your title, describe what happened, pick a dream type, and add tags
1. Your dream is saved automatically
1. Browse, filter, and revisit your dreams anytime

## 🎨 Design

The app uses a **deep sea abyss** aesthetic — inspired by the feeling of diving into something unknown, the way dreams pull you under. The color palette is smoky and desaturated, like colors seen through dark ocean water.

- **Background** — pitch black ocean depths with animated bioluminescent glows
- **Particles** — 55 tiny motes drift upward like sediment floating in deep water
- **Light rays** — soft beams filter down from above, like sunlight reaching the abyss
- **Typography** — Apple SF Pro system font for a clean, modern iOS feel
- **Glass cards** — frosted dark glass panels with subtle teal borders
